    <h2>
        <a href="https://ieeexplore.ieee.org/abstract/document/9185909" target="_blank">
            Genetic Algorithm Applied in UAV's Path Planning
        </a>
    </h2>
    <p><strong>Gustavo de Moura</strong>, Claudio F. M. Toledo</p>
    <p>WCCI CEC - July 2020</p>
    <p>The present paper introduces a hybrid genetic algorithm for path planning problem with obstacle avoidance. The genetic algorithm is combined with Ray Casting (RC) algorithm, where RC is responsible to avoid obstacles and to find safe regions for emergency landing. Thus, the path planning system must deal with a non-convex environment when planning and re-planning trajectories. The system must also work embedded on the UAV running under a Raspberry Pi board. The hybrid method is evaluated over 50 benchmark maps from literature with satisfactory results reported.</p>
</div>
